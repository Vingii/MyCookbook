# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.4.0] - 2025-09-30
### Added
- Added ingredient filtering to recipe tables.

## [1.3.2] - 2025-09-30
### Fixed
- Fixed not being able to delete recipes that were referenced in the meal planner.

## [1.3.1] - 2025-09-21
### Fixed
- Fixed recipe link in planner not working.

## [1.3.0] - 2025-09-20
### Added
- Added a dropdown to the recipe category edit field.
- Added support for middle mouse button click in the recipe browser.

### Changed
- Changed recipe viewer URL to use recipe ID instead of name.

## [1.2.1] - 2025-09-06
### Fixed
- Fixed ingredient not being saved when using Escape.

## [1.2.0] - 2025-09-06
### Fixed
- Added missing localization.
- Fixed planned meals being displayed to everyone.
- Fixed language not being persisted across sessions.

## [1.1.2] - 2025-09-04
### Fixed
- Fixed step description formatting.
- Improved color contrast on step timeline.

## [1.1.1] - 2025-09-04

## [1.1.0] - 2025-09-03
